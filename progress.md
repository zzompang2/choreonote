### 최근 완료
- 공유 뷰어 모바일 반응형 적용
  - grid 레이아웃: 단일 컬럼 5행 (헤더/가로레일/스테이지/플레이어바/타임라인)
  - 세로 레일 → 가로 레일 (에디터와 동일 패턴)
  - 사이드바 → 바텀시트 (슬라이드 업, 드래그 핸들, 바깥 터치 닫기)
  - 캔버스 오버플로우 방지 (minmax(0,1fr) + min-width:0)
  - 데스크톱 레이아웃은 기존 그대로 유지
- 챗봇 1.5단계: FAB(랜딩/노트목록) 도움말 개선
  - FAB 패널에 꿀팁 배너 추가 (페이지별 팁 20초 로테이션)
  - 추천키워드를 body에서 하단 별도 영역으로 분리, FAB에도 실시간 자동완성
  - 팁 데이터를 landing/dashboard/editor/viewer별로 분리
  - 랜딩 전용 FAQ 4개 추가 (무료/기능/설치/오프라인)
  - 스냅/정렬 FAQ 신규 추가, 에디터 추천키워드: 스냅, 교환, 정렬, 단축키
  - 단축키 FAQ → kbd+shortcut-row HTML UI로 변경 (html 플래그 지원)
  - 다른 페이지 context 답변에 사용자 친화적 뱃지 표시 (편집 화면/노트 목록 등)
  - 코드 용어(대시보드/에디터) → UI 용어(노트 목록/편집 화면) 통일
  - 추천키워드 영역 오른쪽 fade-out 힌트 + 스크롤 끝 감지
  - 뷰어 단축키 FAQ 삭제 (에디터와 중복)
  - 팁 텍스트 white-space:nowrap 제거 (모바일 접근성)

### 이전 세션 결정사항 (유효)
- **대열 마켓 방향**: 유저 업로드 마켓(B안), 단일+여러 대형+전환 업로드, 하이브리드 진입(C안: 대시보드 마켓 페이지 + 사이드바 바로가기), 3단계 로드맵
- **댄서 목록 역할 정리**: 중복 편집 기능 제거, 목록은 읽기 전용 + 편집 진입점
