# 진행 상황

## 현재 진행 중: 공유 뷰어 UI 개선

플레이어바 + 사이드바 + grid 레이아웃 구현 완료, 커밋 완료. **브라우저 실제 테스트 미완.**

### 완료한 것
- 플레이어바: 에디터와 동일한 UI (재생/정지/이전대형/다음대형 + 시간)
- 사이드바: 에디터와 동일한 레일 + 패널 구조
  - 댄서 목록 패널 (읽기 전용, 색상+이름)
  - 설정 패널 (라벨 토글, 관객 방향, 음악 로드)
- 레이아웃: 에디터와 동일한 grid 구조 (타임라인이 전체 너비, 사이드바 위에)
- 사이드바 열기/닫기 토글

### 테스트 필요
1. `cd client && npm run dev`
2. `http://localhost:xxxx/#/share/demo001` (Supabase 테스트 데이터)
3. 확인할 것:
   - 사이드바 레일 아이콘 토글 (댄서/설정 전환, 닫기)
   - 댄서 목록 표시
   - 라벨 토글 (번호/이름/없음)
   - 관객 방향 전환
   - 음악 파일 로드
   - 플레이어바 재생/정지/이전/다음 대형
   - 모바일 반응형
