# 진행 상황

## 현재 진행 중: 공유 링크 (재생 전용 뷰어)

코드 작성 + 커밋 완료 (`aaf2090`). 뷰어 UI 리팩토링 진행 중 (사이드바 재활용, 툴바 정리).

### 뷰어 UI 변경사항 (미커밋)
- `.viewer__toolbar`, `.viewer__player-bar` 등 커스텀 요소 제거 → 에디터 사이드바 컴포넌트 재활용
- `.viewer__content` 래퍼 추가 (flex 레이아웃)
- `.dancer-item__name-label` 스타일 추가

### 테스트 방법
1. `cd client && npm run dev`
2. 에디터에서 노트 생성 → 무대설정 > "공유 링크 복사" 클릭
3. 새 탭에서 복사된 URL 열기 → 재생 전용 뷰어 확인

### 확인할 것
- 뷰어 렌더링 정상 동작
- 재생/정지, 타임라인 마커 드래그, 대형 선택
- 댄서 목록 사이드바 표시
- 모바일 반응형
