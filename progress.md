# 진행 상황

## 현재 진행 중: 공유 링크 (재생 전용 뷰어)

코드 작성 + 커밋 완료 (`aaf2090`). **브라우저 실제 테스트 미완.**

테스트 방법:
1. `cd client && npm run dev`
2. 에디터에서 노트 생성 → 무대설정 > "공유 링크 복사" 클릭
3. 새 탭에서 복사된 URL 열기 → 재생 전용 뷰어 확인
4. 또는 `http://localhost:5173/#/share/demo001` (Supabase에 테스트 데이터 있음)

확인할 것:
- 뷰어 렌더링 정상 동작
- 재생/정지, 타임라인 마커 드래그, 대형 선택
- 줌 배지 위치 (스테이지 우하단에 있어야 함)
- 댄서 라벨 토글 (번호/이름/없음)
- 관객 방향 토글 (좌표 180° 회전 포함)
- 전환 구간 이동 경로 표시
- 모바일 반응형
