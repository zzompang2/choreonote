### 최근 완료 (2026-04-17)
- 대열 마켓 1.1~1.5단계 전면 개선
  - **업로드 3스텝 플로우**: 노트 선택 → 대형 갤러리(3열 그리드) 범위 선택 → 정보 입력
  - **연속 범위 선택**: 클릭=단일 확정, 추가 클릭=범위 확장, hover 미리보기 (최대 5개)
  - **느낌 태그 8종**: 칼군무/부드러운/대칭/역동적/웅장한/밀집/흩어짐/귀여운 (최소 1개 필수)
  - **대표 썸네일 선택**: 스텝3에서 선택한 대형 중 대표 지정
  - **설명란 제거**, 제목은 유지 (랜덤 플레이스홀더 예시 8종)
  - **퇴장 댄서 처리**: dancer_count에서 무대 밖 댄서 제외 + 썸네일에서 hideOffstage
  - **모달 레이아웃**: top/body/bottom 3분할 (헤더·버튼 고정, 콘텐츠만 스크롤), 높이 70vh 고정
  - **스텝 인디케이터**: 1-2-3 번호, 이전(왼쪽)/다음(오른쪽) 버튼 배치
  - **disabled 버튼 UI**: btn--primary:disabled opacity 0.35
  - **관객 방향 토글**: 필터 바에 👥 버튼, 모든 썸네일/미리보기 좌표 180도 반전
  - **정확한 인원수 필터**: 범위 필터 → 개별 숫자(2~8, 9+)
  - **태그 필터링**: 복수 선택, Supabase JSONB contains 쿼리
  - **카드 태그 표시**: 마켓 카드와 상세 모달에 태그 칩 표시

### 결정사항 (유효)
- **마켓 업로드**: 체크박스 대신 연속 범위 선택 (단일 or 최대 5개 연속)
- **태그 vs 설명**: 설명란 제거, 느낌 태그(8종)로 대체. 모양 태그는 빼고 썸네일이 담당
- **제목 유지**: 모양 태그 없으므로 검색·표현 수단으로 제목 필요
- **인증 정책**: 공유 링크는 익명, 마켓 가져오기/업로드만 로그인
- **용어 규칙**: 대형(Formation), 동선 구간(전환 영역), 동선(이동 경로)
- **클라우드 저장**: 음악 미포함, IndexedDB primary, 서버는 백업
